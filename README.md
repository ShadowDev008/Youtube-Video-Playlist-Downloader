# Youtube-Video-Playlist-Downloader
Download YouTube Videos and Playlist in Mp4 or Mp3 format

# Features

- 🎬 **MP4 (Video) & MP3 (Audio) Download** in höchster Qualität
- 📂 **Playlist-Download** mit Einzelfortschritt und Batch-Historie
- 📚 **Library-Tab** mit:
  - Download-Historie
  - Doppelklick zum Öffnen
  - Kontextmenü
- ✂️ **Trimmer-Tab** zum Zuschneiden von MP4/MP3 via `ffmpeg`
  - Kein Re-Encoding
  - Schnelles verlustfreies Schneiden
- 📦 **ffmpeg direkt in der EXE gebündelt**
  - Keine externe Installation erforderlich
- 💾 **Portable Version**
  - Funktioniert direkt vom USB-Stick
- 📝 **Download-Historie als JSON**
  - Wird automatisch neben der EXE gespeichert

---

# Technologien

- Python
- yt-dlp
- ffmpeg
- Custom GUI

---

# Hinweise

- Keine zusätzliche Installation von `ffmpeg` notwendig
- Internetverbindung für Downloads erforderlich
- Unterstützt Video- und Audio-Downloads in hoher Qualität
